The Form Element

The form element represents a document section containing interactive controls for submitting information.

Attribute - method="get" (default)

Form data appended to the action URL with a ? separator.

<form action="#" method="get">
    <label for="search-bar-get">Search</label>
    <input name="search-bar" id="search-bar-get" type="search">
    <button>Submit</button>
</form>

Attribute - method="post" (default)

Form data sent as the request body.

<form action="#" method="post">
    <label for="search-bar-post">Search</label>
    <input name="search-bar" id="search-bar-post" type="search">
    <button>Submit</button>
</form>
